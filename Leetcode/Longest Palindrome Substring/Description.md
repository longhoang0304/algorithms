### Desc

Cho chuỗi S, tìm chuỗi con dài nhất của S sao cho chuỗi con này là một chuỗi palindrome.

Ví dụ:
S = abccc -> chuỗi con dài nhất là ccc

### Input
Chuỗi s

### Output
Chuỗi con dài nhất của S là chuỗi palindrome

### Sample Input
```
abcdbbfcba
```

### Sample Output
```
bb
```
