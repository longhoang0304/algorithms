### Url
https://leetcode.com/problems/3sum/

### Desc

Cho 1 Arr mảng các số nguyên. Trả về 1 mảng, gồm 3 số a, b, c sao cho a + b + c = 0. Các cặp (a, b, c) không được trùng nhau

### Input
int Arr[]

20 <= nums.length <= 3000
-10^5 <= nums[i] <= 10^5

### Output
Mảng chứa các cặp 3 số (a, b, c) sao cho a + b + c = 0

### Sample Input
```
Arr = [-1,0,1,2,-1,-4]
```

### Sample Output
```
[[-1,-1,2],[-1,0,1]]
```
